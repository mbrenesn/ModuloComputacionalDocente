# Modulo Computacional Docente de la Escuela de Física
Materiales y recursos para el módulo computacional docente de la Escuela de Física UCR.

# Para clonar este repositorio

En alguna ubicación (e.g. ~),

`git clone https://github.com/mbrenesn/ModuloComputacionalDocente.git`

# Actualizaciones

Conforme el curso avanza; nuevos archivos, códigos y recursos se irán añadiendo y/o modificando.
Para mantener el repositorio actualizado, ejecutar el siguiente comando paulatinamente

`git pull origin main`

# Instalar Miniconda
`mkdir -p ~/miniconda3`

`wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh -O ~/miniconda3/miniconda.sh`

`bash ~/miniconda3/miniconda.sh -b -u -p ~/miniconda3`

`rm -rf ~/miniconda3/miniconda.sh`

Luego,
`~/miniconda3/bin/conda init bash`
